Wikipedia Article Scraper, Personal Project

This is a fun python project I did. It takes a link to a Wikipedia article as input and will give a related article as output. This was done using the Python libraries requests and random; the function gathers all the links (a tags) from the input article page and follows a random link. It will do this a specified number of times, resulting in an article which is more or less related to the input article. This was a fun starter project which was also a challenge because I had to learn about recursion.
