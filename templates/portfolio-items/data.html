Data Analytics Intern, Environmental Data Initiative, Baylor University

Working with Environmental Data Initiative this past summer, I have been curating lake and stream datasets for use in the EDI data portal, an open source platform for researchers to view and work with large environmental datasets. I have been performing a variety of ETL and data analysis tasks, such as data cleaning and metadata wrangling, and I have been creating data visualization tools and dashboards using RShiny. This work has been impactful as the data packages and visualization tools will be used by dozens of researchers. I have enjoyed the opportunity to learn more about data manipulation and presentation as well as the experience working in a team and conducting peer code reviews.

Programming Research Assistant, Todd-Brown Lab, Environmental Engineering Department, UF

At the Todd-Brown Lab I worked heavily with computational tools such as R to analyze, visualize, and manage carbon stock models, which inform scholars and policymakers how vital ecosystems will respond to climate change. I gained experience with data manipulation, statistical analysis, and model visualization. I also performed code reviews / refactoring and code documentation for our model. One of the most interesting projects that I worked on was a sensitivity analysis, which involved running model simulations thousands of times and calculating the numerical gradient across the model space to determine which parameters were more sensitive than others. This experience was very valuable to me as I learned a lot about programming and working in a team, and I gained the mindset to solve complex problems in an efficient way.

Green Riders Infographic, Tableau Project

I wanted to create a project in Tableau to hone my data visualization experience. I decided to create an infographic to showcase the transcontinental bike volunteering tour I was a part of in 2019. It is nice to have a graphic to commemorate the good deeds that we did as well as to remember all the locations we visited. The data gathering was mostly done manually using excel, as a lot of my records from the trip were either hand-written or on another spreadsheet. The rest of the project was done in Tableau.
