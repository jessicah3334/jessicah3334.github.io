Intern / Research Fellow, Environmental Data Initiative, Baylor University

Working with Environmental Data Initiative this past summer, I have been curating lake and stream datasets for use in the EDI data portal, an open source platform for researchers to view and work with large environmental datasets. I have been performing a variety of ETL tasks and metadata wrangling, as well as creating visualization tools using RShiny. Restoration Analysis of the Captain Jack Mill Superfund Site, Groundwater Hydrology, UF

For this project, I did a contaminant and restoration analysis of the Captain Jack Mill Superfund Site in Boulder County, CO. This involved creating a site model, conducting relevant calculations (mass flux, cost estimate, remediation scenarios), and modeling the contaminant plume using Interactive Groundwater (IGW). This was a great project for learning remediation strategies because I was able to create my own solution strategy and compare it with the strategy that the engineers employed in real life.

Todd-Brown Lab, Environmental Engineering Department, UF

At the Todd-Brown Lab I worked heavily with computational tools such as R to analyze, visualize, and manage carbon stock models, which inform scholars and policymakers how vital ecosystems will respond to climate change. I also gained experience writing manuscripts and collaborating with people from different professional backgrounds. Working with the Marsh Equilibrium and Grow-Burn-Decay carbon models, I crafted a GUI (graphic user interface), coauthored a manuscript, created model visualization tools using R packages, and helped conduct a sensitivity analysis. This experience has been truly valuable because I gained the tenacity to work on difficult problems, experience working with many different R packages, Markdown, and Git, and the mindset to solve problems efficiently.

Lake Caroline Water Treatment Plant, Potable Water System Design

This was my senior design project for Environmental Engineering. I worked in a team of 6 to design a full-scale water treatment facility which included plant elements such as design-drawings, historical analysis, pipe network design, chemical treatment doses, and tank sizing. I am proud of this project because our team put a ton of effort into perfecting our final design report, which ended up being over 100 pages. I enjoyed the practicality and satisfaction that came from completing this project. The biggest challenge that we faced was the hands-off approach that our professor took, which we solved by pooling together as a team and making the best decisions with the information we had.

Wikipedia Article Scraper, Personal Project

This is a fun python project I did. It takes a link to a Wikipedia article as input and will give a related article as output. This was done using the python libraries requests and random; the function gathers all the links (a tags) on the input article page and follows a random one. It will do this a specified number of times, resulting in an article which is more or less related to the input article. This was a fun starter project which was also a challenge because I had to learn about recursion.
